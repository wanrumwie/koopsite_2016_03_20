Результати тестування попереднього перегляду файлів
різних типів на різних браузерах/серверах

Chrome
        Django server       Apache server       pythonanywhere.com
Тип     /view   /preview    /view   /preview    /view   /preview
.jpg    +       +           +       +
.pdf    +       -(4)        +       +
.txt    +(1)    -(2)        +(1)    +
.tif    -(2)    (3)         -(2)    (3)
.doc    -(2)    (3)         -(2)    (3)
.docx   -(2)    (3)         -(2)    (3)
.xls    -(2)    (3)         -(2)    (3)
.xlsx   -(2)    (3)         -(2)    (3)

FireFox
        Django server       Apache server
Тип     /view   /preview    /view   /preview
.jpg    +       +           +       +
.pdf    +       -(6)        +       +
.txt    +(1)    -(6)        +(1)    +
.tif    +(5)    (3)         +(5)    (3)
.doc    +(5)    (3)         +(5)    (3)
.docx   +(5)    (3)         +(5)    (3)
.xls    +(5)    (3)         +(5)    (3)
.xlsx   +(5)    (3)         +(5)    (3)

Примітки:
+(1) - текст виводиться з неправильним кодуванням кирилиці
-(2) - завантажується без попередження
 (3) - template має обмеження на ці типи файлів
-(4) - показує біле пусте місце, але не вантажить pdf
+(5) - FireFox перепитує, що зробити з файлом
        і для перегляду пропонує коректний застосунок
-(6) - показує чорне пусте місце з повідомленням:
        A plugin is needed to display this content
